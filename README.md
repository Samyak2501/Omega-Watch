# Premium Watch Scroll Experience

A premium scroll-based luxury watch showcase built with modern frontend technologies, cinematic animations, and immersive storytelling.

## ✨ Overview

This project focuses on creating a high-end digital product experience where motion, typography, and visuals work together to tell a story.

The website features a luxury watch presented through scroll-driven interactions, premium design aesthetics, and responsive layouts.

## 🚀 Features

* Smooth scroll-based storytelling
* Cinematic watch showcase
* Premium UI/UX design
* Responsive across devices
* Scroll-controlled animations
* Interactive content transitions
* Performance-optimized frontend
* Modern luxury-inspired visuals

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
* GSAP
* ScrollTrigger
* Lenis
